windows环境搭建github.io博客

软件安装

安装nodejs(包含了npm工具)

Download and install Windows Installer (.msi) from

安装hexo

npm install hexo-cli -g

安装git

git download

配置git的ssh公钥
git config --global user.name zd  
git config --global user.email dkd1168@126.com  

ssh-keygen -t rsa -C dkd1168@126.com

将公钥id_rsa.pub的内容拷贝到github页面的sshkeys

安装hexo的git推送器

npm install hexo-git-backup --save

hexo 创建项目

hexo init zd_blog_test  
cd PROJECT_NAME & npm install

确认项目是否成功, 启动本地服务器, 访问http://localhost:4000 

hexo server

hexo部署到github.io

hexo有3个作用

  1. hexo new filename 生成md文件
  2. hexo g 将md文件转为html
  3. hexo d 将html文件推送到github.io
  4. 2,3可以合为一步hexo g –d

创建github.io账号

给仓库起个名称，注意！注意！注意！

仓库名称格式： 用户名.github.io 

https://github.com/zjqs007/zjqs007.github.io

配置hexo的_config.yml

# DeploymentV  
## Docs: https://hexo.io/docs/one-command-deployment  
deploy:  
  type: git  
  repo: git@github.com:zjqs007/zjqs007.github.io.git  
  branch: master  
  message: update

发布到github.io

  1. 新建博客
    hexo new zd-fisrt-blog

找到zd_blog_text/source/_posts/zd-fisrt-blog.md, 使用vscode打开并且编辑
